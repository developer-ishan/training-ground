# RPC Practical Examples

This directory contains practical, runnable examples of RPC implementations.

## Examples Included

1. **gRPC Example (Python)** - Complete gRPC server and client
2. **JSON-RPC Example (Node.js)** - Simple JSON-RPC implementation
3. **gRPC Example (Go)** - Go-based gRPC server with streaming

## Prerequisites

### For gRPC Python Example
```bash
pip install grpcio grpcio-tools
```

### For JSON-RPC Node.js Example
```bash
npm install
```

### For gRPC Go Example
```bash
go install google.golang.org/protobuf/cmd/protoc-gen-go@latest
go install google.golang.org/grpc/cmd/protoc-gen-go-grpc@latest
```

## Running Examples

See individual README files in each example directory for specific instructions.
